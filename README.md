# PureGlow-Decodelabs
PureGlow is a modern, fully responsive beauty and skincare website designed with a clean and elegant user interface. The website showcases premium skincare, makeup, hair care, fragrances, and beauty products with an engaging shopping experience.
Features
Responsive design for desktop, tablet, and mobile devices
Modern hero section with call-to-action buttons
Product categories and featured products
New arrivals section
Limited-time promotional banner
Customer testimonials
Beauty tips and blog section
Newsletter subscription form
Contact section with location, phone, and email
Smooth scrolling navigation
Clean, user-friendly, and attractive UI
Technologies Used
HTML5
CSS3
JavaScript
Font Awesome
Google Fonts
Purpose

This project was created to showcase modern front-end web development skills and responsive UI design. It demonstrates the ability to build an attractive e-commerce landing page for beauty and skincare brands.
